kokaq

kokaq is an open source, distributed, cloud-native priority queue implementation, designed to solve the long-standing gap of native priority handling in cloud messaging systems. It’s built for high availability, low latency, and intelligent resource allocation — empowering mission-critical workflows with dynamic, weight-based prioritization.

Get Started → GitHub Documentation

Highest Priority First

Ensures that the most important messages are always processed first, deterministically.

Non Discrete Priorities

Fine-grained control using continuous, weight-based priority values instead of fixed levels.

Cloud-Native Architecture

Built from the ground up on Kubernetes and other modern cloud environments

Performance

Designed to handle latency-sensitive and real-time workloads with minimal delay and strong reliability.

Service Integration Ready

Seamlessly connects with Azure Functions, AWS Lambda, Cloud Functions.

Observability and Insights

Built-in support for real-time monitoring through OpenTelemetry based collectors.

kokaq is open-source

kokaq and all of its dependent packages, libraries, and modules are open-sourced under the MIT License.

We're building kokaq to be a fast, scalable and highly usable priority queue that is easy to deploy and quick to integrate. We're open-sourcing it to allow the community to contribute and make it better.

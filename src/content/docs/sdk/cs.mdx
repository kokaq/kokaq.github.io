The [kokaq .NET SDK](https://github.com/kokaq/kokaq-client) is the official .NET client for kokaq.

> Make sure that the kokaq server is running before you start using the SDK
> and following this guide.

## Installation

To install the kokaq .NET SDK, via NuGet Package Manager or dotnet:

```bash
## NuGet
Install-Package Kokaq.Client

## dotnet
dotnet add package Kokaq.Client
```

The [kokaq Go SDK](https://github.com/kokaq/client) is the official Go client for `kokaq`.

> Make sure that the kokaq server is running before you start using the SDK
> and following this guide.

## Installation

To install the kokaq Go SDK, use `go get`:

```bash
go get github.com/kokaq/kokaq-go
```
